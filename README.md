# SPU0410LR
The Converter board of Kicad Data for SPU0410LR5H

## 概要

Knowles社製の[アナログ出力MEMSマイクSPU0410LR5H][1]を2.54mmピッチへ変換する基板です。  
可聴音から超音波(100Hz～80kHz)までの幅広い帯域を高感度にセンシングすることが可能です。  
超音波センサや超音波通信などに使用可能です。  
マイク穴は基板裏面の丸印部分です。  
電源電圧1.5V~3.6V
